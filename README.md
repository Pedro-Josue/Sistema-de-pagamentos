## Descrição do Projeto
Aplicação Console em C# que simula um sistema de pagamentos com duas opções:
- Pagamento via Cartão
- Pagamento via Boleto

O sistema coleta o valor e os dados específicos, processa o pagamento e exibe um resumo.

## Funcionalidades
- Menu principal com repetição até a opção sair
- Pagamento via cartão
- Pagamento via boleto
- Tratamento de entradas inválidas
- Aceita valores monetários com vírgula ou ponto

## Tecnologias
- C#
- .NET Console Application

## Como executar
```bash
dotnet run
```
